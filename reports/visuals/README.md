EDA Visualizations.

This folder contains charts, graphs, and screenshots generated during
the Exploratory Data Analysis (EDA) of the IVF dataset.

Visuals include:
- Distribution plots
- Missing value analysis charts
- Outlier detection visuals
- Univariate and bivariate analysis graphs
- Power BI screenshots (if applicable)

Dashboard KPIs:
- Clinical Pregnancy Rate (%)
- Live Birth Rate (%)
- Treatment Cycle Analysis
- Age Group Comparison


These visuals support the insights documented in the EDA report.
