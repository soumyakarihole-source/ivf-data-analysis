Exploratory Data Analysis (EDA) Report.

This report summarizes the statistical and visual analysis performed on the IVF dataset
to understand data quality, distributions, patterns, and relationships between variables.

Key EDA activities include:
- Data understanding and structure review
- Missing value analysis
- Descriptive statistics (mean, median, variance, skewness, kurtosis)
- Outlier detection
- Univariate and bivariate analysis
- Initial insights related to IVF outcomes

The detailed EDA findings are documented in the accompanying report files.
